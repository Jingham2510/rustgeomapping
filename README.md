# rustgeomapping
Geotechnical mapping tools using RGBD cameras

Originally a part of rustbot_cntrl ([here](https://github.com/Jingham2510/rustbot_cntrl)) - now transferred out so that it can be used in other systems in the lab.

REQUIREMENTS:
-If using the "sense" feature, this crate requires an installation of the realsense SDK such that the rust bindings can be used
